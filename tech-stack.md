# Tech Stack: generate-iron-stock

## Frontend
- **Framework**: Next.js (TypeScript) - Using App Router for modern, efficient routing.
- **PWA Capabilities**: Implementation as a Progressive Web App (PWA) using `next-pwa` to allow "Add to Home Screen" and offline capabilities.
- **Styling**: Vanilla CSS (Mobile-first, responsive, and high-visibility focus)
- **UI Components**: Headless UI or Radix UI (for accessible primitives)
- **Icons**: Lucide React (Large, clear stroke icons)

## Backend
- **Runtime**: Node.js (Integrated within Next.js API Routes or a separate Express server if complex logic is required)
- **Database**: PostgreSQL (FIFO-ready schema)

## Infrastructure & Tools
- Gemini CLI with Conductor Extension for project orchestration.
- Git for version control.
