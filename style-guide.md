# generate-iron-stock 設計風格手冊 (Style Guide) v1.2

## 核心願景
針對 50 歲以上鋼鐵產業用戶優化的「行動優先」設計語言，強調高對比、超大點擊區域、以及明確的狀態反饋。

---

## 1. 視覺識別 (Visual Identity)
| 類別 | 顏色代碼 | 用途 |
| :--- | :--- | :--- |
| **主色調 (Primary Blue)** | `#2563eb` | 確認按鈕、導航、主要動作、主要圖示 |
| **成本色 (Cost Orange)** | `#c2410c` | 成本數值、金額輸入框、獲利提醒 |
| **表面色 (Surface Gray)** | `#f8fafc` | 卡片背景、頁面分區、輸入框底色 |
| **警示色 (Danger Red)** | `#ef4444` | 刪除按鈕、錯誤提示、警告 |

---

## 2. 字體規範 (Typography)
*   **全域字體:** 優先使用系統預設黑體，確保在行動裝置上的渲染清晰。
*   **頁面標題 (Senior-XL):** `26px` / `700 (Bold)` / `Slate-900`
*   **區塊標題 (Senior-LG):** `22px` / `600 (Semi-Bold)` / `Slate-800`
*   **內文正文 (Base):** `18px` / `Normal` / `Slate-700`
*   **輔助標籤 (Label):** `14px` / `700 (Bold)` / `Slate-600` / 搭配圖示

---

## 3. 元件標準 (Component Standards)
### 3.1 觸控與尺寸
*   **最小觸控目標:** 所有可點擊元件（按鈕、輸入框）高度至少為 `52px`。
*   **圓角規範:** 大元件（卡片、主按鈕）使用 `24px (Rounded-3xl)`。

### 3.2 邊框與狀態提醒
*   **預設狀態:** `2px solid #dbeafe` (淡藍)。
*   **懸停與焦點 (Hover/Focus):** 邊框加深至 `2px solid #93c5fd` 並疊加 `shadow-md` 提升層次。
*   **金額專用:** 使用 `2px solid #fed7aa` (淡橘) 與淺橘背景 `#fff7ed`。

### 3.3 互動回饋
*   **按壓感:** 點擊按鈕具備 `active:scale-95` 或 `active:scale-[0.98]` 的物理縮放效果。
*   **圖示輔助:** 欄位必須搭配圖示 (Icon)，不可僅依賴文字。

### 3.4 動作按鈕佈局 (Action Button Placement)
*   **新增資料按鈕:** 統一放置於 **資料清單末尾 (List End)**。
    *   **理由:** 符合「由上而下」的操作流，填完現有資料後自然銜接新增動作。
    *   **視覺:** 必須為 **100% 寬度**，建議使用 **虛線邊框 (Dashed Border)** 與 **淺色背景 (Surface)**。
*   **全域存檔按鈕:** 統一固定於 **螢幕底部 (Sticky Bottom)**，需與導航條保持適當層級感。

---

## 4. 佈局規範 (Layout Guidelines)
*   **行動優先架構:** 內容最大寬度限制為 `448px` (max-w-md)，於大螢幕自動居中。
*   **單一垂直流:** 嚴禁水平捲動。所有操作邏輯應一律由上至下展開。
*   **底部安全距離:** 頁面底部必須保留至少 `128px (pb-32)` 的空白，避免內容被固定導航條擋住。
*   **操作防呆:** 禁止使用側滑 (Swipe) 等隱藏手勢，所有功能按鈕必須直接可見。

---

## 5. 圖示標準化 (Iconography)
*   **圖示庫:** 使用 `Lucide React`。
*   **筆畫寬度:** 統一設置為 `Stroke Width: 2` 或 `3`，增強辨識度。
*   **尺寸:** 主要動作圖示為 `28px`，輔助圖示為 `16px` - `20px`。
