# Product Guidelines: generate-iron-stock

## Visual Style (Mobile & Senior-Friendly Focus)
- **Mobile-First**: The UI is primarily designed for portrait smartphone screens.
- **Aesthetic**: Functional, clean, and highly legible. Avoid cluttered interfaces.
- **Typography**: Large, clear font sizes (minimum 18px-20px for mobile body text). High-contrast sans-serif fonts.
- **Color Palette**: High-contrast color combinations. Use large, distinct primary-colored buttons for core actions.
- **Layout**: Single-column layout optimized for vertical scrolling. Large tap targets (minimum 48x48 pixels).

## Interactions
- **One-Handed Operation**: Place frequent actions within easy reach of the thumb.
- **Simplified Navigation**: Bottom navigation bar for core features (Restock, Ship, History). Avoid complex gestures.
- **Explicit Labels**: Use large text labels next to or below every icon.
- **Visual & Haptic Feedback**: Provide clear confirmation and haptic feedback for successful actions.
- **Form Design**: Use large, easy-to-tap input fields. Minimize typing by using selectors or simple +/- buttons for quantity.

## Accessibility
- Full keyboard support and clear visual focus indicators.
- Screen reader compatibility with descriptive ARIA labels.
- Support for PWA "Add to Home Screen" for an app-like experience.
